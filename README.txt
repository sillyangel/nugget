# Nugget Mod

Nugget Mod is a Minecraft mod that adds "" to the game, the Nugget. The Nugget is a powerful item that can be used to craft new tools, weapons, and armor. The mod also adds new ores to the game, including Nugget Ore, which can be mined to obtain Nuggets. The mod is still in development, but you can download the latest version from the releases page.
